
A Morse Code Iambic Keyer using a Wii Nunchuk

======

Pins:

I2C SCL is PIN A5

I2C SDA is PIN A4

LED/relay output is PIN D4

Audio output is PIN D8

======

Libraries used:

Arduino.h
Wire.h

======

WPM Configuration:

Press the Wii joystick up/down to change the WPM setting.

The WPM range is from 10WPM to 25WPM.
The initial WPM setting is 15WPM.

======

Dit/Dah Key Assignment:

Press the Wii joystick left/right to swap/unswap the dit/dah key assignment
