
A Morse code keyer with USB keyboard and LCD

======

Pins:

I2C SCL is PIN A5

I2C SDA is PIN A4

LED/relay output is PIN D5

Audio output is PIN D8

======

Libraries used:

hidboot.h
SPI.h
LiquidCrystal_I2C.h

======

WPM Configuration:

Press ESC key to enter/exit config mode.
While in config mode the up/down arrow keys change the WPM setting.
The new WPM value is displayed on the LCD display.

The WPM range is from 10WPM to 25WPM.
The initial WPM setting is 15WPM.
